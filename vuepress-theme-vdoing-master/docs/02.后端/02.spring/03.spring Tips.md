1. snapshot 是快照版本，GA是稳定版本
2. spring 5.0 如果缺少日志包，直接导入（spring-jcl-版本号）包即可，无需导入commons-log
3. spring2.5开始支持注解的使用，spring3.0开始支持使用javaConfig配置spring，spring4.0支持springboot1.0，springboot完全使用javaConfig
